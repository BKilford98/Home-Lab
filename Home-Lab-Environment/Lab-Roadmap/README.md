# Home Lab Roadmap

This document outlines planned improvements and learning goals for my home lab environment.

## Short-Term Goals
- Add more users and explore user management
- Explore and learn reconnaissance tools
- Learning to identify attack surfaces

## Medium-Term Goals
- Apply Group Policies
- Test security scenarios using Metasploitable
- Monitor network traffic using PFsense logging
- Complete Active Directory installation and domain join

## Long-Term Goals
- Explore incident response exercises
- Add more complex network topology
- Integrate additional Linux/Windows clients for testing
